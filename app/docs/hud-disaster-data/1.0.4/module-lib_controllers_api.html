(inner) /applicants/:queryType(:queryType, disasters, state, localeType, locales, cols):JSON

router.get('/applicants/:queryType')

Parameters
:queryType:string

either summary or export

disasters:string

a comma separated list of disaster id's

state:string

a state id

localeType:string

a type of geographic area

locales:string

a comma separated list of geographic area's, requires specifying geoName

cols:string

a comma separated list of columns to be returned, or to be returned with the summed values

Returns
Type
:JSON
  • household level data or summary data, filtered by other parameters
Example
// returns JSON with all columns of household level data for disasters 4311,5130, and 5182
 get /applicants/export?disasters=4311,5130,5182
// returns JSON with summarized columns numberOfRecords (automatically added to returned columns for summary), total_dmge_amnt, and hud_unmt_need_amnt of household level data for Wisconsin cities Madison and Monroe
 get /applicants/summary?states=WI&localeType=city&locales=Madison,Monroe&cols=total_dmge_amnt,hud_unmt_need_amnt

(inner) /disasternumber/:qry(:qry)

router.get('/disasternumber/:qry')
queries FEMA API (https://www.fema.gov/api/open/v1), and returns disaster data for specific disasters

Parameters
:qry:string

a comma separated list of disaster id's

Example
// returns disaster array that is limited by the specified disasters (in this case, the Wisconsin cities Madison and Monroe)
 get /disasternumber/DR-4311-UT,FM-5130-UT,FM-5182-WA

(inner) /disasterquery/:qry(:qry)

router.get('/disasterquery/:qry')
queries FEMA API (https://www.fema.gov/api/open/v1), and returns disaster data

Parameters
:qry:string

one or more of: the disaster type, the disaster number, the state. If more than one, needs to start with disaster type, followed by a dash (-), followed by diaster number, optionally followed by another dash and the state abbreviation.

Example
// returns array of disasters (in this case, of disasterType DR)
 get /disasterquery/DR

// returns array of disasters (in this case, of disasterNumber 4311)
 get /disasterquery/4311
